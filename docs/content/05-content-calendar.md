# Content Calendar & Pipeline — Phase 2 Documentary Funnel

> **Source:** RelatiV GTM & Product Strategy Summary, v1.0 (2026-06-11). Section 2 — "Content-Led Distribution & The Documentary Funnel."
> **Window:** 16 weeks (4 months). Tunable.
> **Assumption:** 8–10 alpha campaigns complete in Days 1–15 (Phase 1). Calendar starts at **Day 16**, end of Phase 1.

---

## Pipeline Overview

```
PHASE 1 (Days 1–15)         PHASE 2 (Days 16–~120)
───────────────             ──────────────────────────────────
8–10 alpha campaigns  ──▶  8–10 Case-Study Doc Shorts
       │                          │
       │                          ▼
       │                   Case-Study Doc #1 (long-form, YouTube)
       │                          │
       │                          ▼
       │                   3 Cultural Deep-Dive Docs (long-form)
       │                          │
       │                          ▼
       └──────────────────▶ 24–36 short-form tease clips (cross-platform)
```

**Constraint:** every long-form doc fans out into 8–12 short-form clips. The short-form clips are how the long-form gets discovered.

## Week-by-Week Calendar (16 weeks)

### Weeks 1–2: Alpha doc post-mortem & funnel priming
- [ ] **Day 16 — Day 22:** Day-1 doc drop. Publish **Case-Study Doc #1** (10–14 min) — alpha cohort's strongest result.
- [ ] **Day 16 — Day 22:** Publish 8–10 short-form clips from Doc #1, staggered across the week.
- [ ] **Day 17–22:** Outreach: tag the alpha client, share the doc, request their cross-post. (This is the "advocate" loop from the GTM doc.)
- [ ] **Day 18:** Internal retro: which clips performed best? Use the data to pick the **production template** for the next case-study doc.

**Goal:** prove the alpha-doc-to-short-form-fanout motion works *before* committing to the 8–10 case-study cadence.

### Weeks 3–6: Case-study doc cadence (alpha cohort)
- [ ] **1 case-study doc per week** for 4 weeks (4 docs total). With 8–10 alpha campaigns, this covers ~half the cohort; the other half go in Weeks 9–12.
- [ ] Each doc: 10–14 min long-form + 8–10 short-form clips.
- [ ] **Mid-cadence retro (Week 5):** which of the 4 docs hit hardest? Use that template as the cultural-doc outline.

> **Decision point (Week 6):** which cultural doc to lead with — Playboi Carti, Black Boy Max, or Adin Ross 2021. **This is a taste call.** Recommend the Adin Ross 2021 doc for *RelatiV-pipeline fit* (most explicit mirror of the AI product), but the call is yours.

### Weeks 7–8: Pre-production for cultural docs
- [ ] Lock the 3 cultural doc outlines (`02`, `03`, `04` in `/docs/content/`).
- [ ] Sourcing sprint for the lead doc — pull primary assets, fair-use check.
- [ ] Cultural-sensitivity reviewer identified per doc.
- [ ] Thumbnail + title A/B variants prepared.
- [ ] **Series title locked** ("Distribution Without Permission" is a working title — confirm or replace).

### Weeks 9–10: Cultural Doc #1 (lead)
- [ ] **Publish long-form doc** (Day 64-ish).
- [ ] 8–10 short-form tease clips staggered across 7–10 days.
- [ ] Outreach to creator-economy / music-TikTok commentators for cross-coverage.
- [ ] Mid-doc retro: what did the audience reaction tell us to adjust in Docs #2 and #3?

### Weeks 11–12: Cultural Doc #2 + remaining case-study docs
- [ ] Publish Cultural Doc #2.
- [ ] Use Weeks 11–12 to catch up on any remaining alpha case-study docs.
- [ ] Combined short-form cadence: cultural doc clips + case-study clips running in parallel.

### Weeks 13–14: Cultural Doc #3 + series finale
- [ ] Publish Cultural Doc #3.
- [ ] **Series finale clip** — a single long-form that ties all 3 cultural docs together with a RelatiV-pipeline overlay. This is the **single most valuable asset of Phase 2** — it's the "here's the pattern across all three" doc that becomes the SaaS funnel's hero piece.

### Weeks 15–16: Phase 2 retro & Phase 3 prep
- [ ] Aggregate analytics: which doc + which short-form clip set drove the most RelatiV signups / trial requests / waitlist adds?
- [ ] Repurpose the top-performing long-form into a 60-second RelatiV brand spot.
- [ ] **Phase 3 readiness check:** is the SaaS stable enough to start the D2C affiliate layer pilot? (Gating decision.)

## Content Volume Targets (16-week window)

| Asset type | Volume | Cadence |
|---|---|---|
| Case-study docs (long-form) | 8–10 | 1/week, Weeks 3–12 |
| Cultural deep-dive docs (long-form) | 3 + 1 finale | Weeks 9, 11, 13, 14 |
| Short-form tease clips | 96–140 (10–14/doc × 11 docs) | Daily cadence from Week 1 |
| Cross-posted quote-cuts (X) | 30+ | 2–3/week |
| Newsletter / Substack issues | 8 | Bi-weekly |

**Realistic volume check:** 96–140 short-form clips in 16 weeks is ~6–9 clips/week sustained. With the RelatiV AI pipeline, this should be feasible. With manual editing it's not. The pipeline proves itself by hitting this volume.

## Funnel Mechanics — How Content Drives Signups

```
Short-form clip views
        │
        ▼
Profile / channel click (profile pic, link in bio)
        │
        ▼
Long-form doc watched (10–14 min, hook retention check at 1:30)
        │
        ▼
End-of-doc CTA → RelatiV demo / alpha cohort application
        │
        ▼
Trial signup (10–15 day, no-cost)
        │
        ▼
Conversion (alpha → paid)
        │
        ▼
Case-study doc with the new paying customer → loop restart
```

**This is the self-sustaining infinite funnel from the GTM doc.** Each paying customer feeds the next case-study doc. Each doc feeds the next short-form clip cycle.

## Risks & Mitigations

| Risk | Mitigation |
|---|---|
| Cultural doc misses the cultural moment | Time the publish date against anniversaries / comebacks / news cycles |
| Short-form cadence slips | Use RelatiV itself for the fanout — dogfood the product for the product's own content |
| Alpha case-study docs are too dry to be the funnel's lead | Use the cultural docs as the lead asset; case-study docs are the *credibility proof* underneath |
| Carti / Adin Ross docs have legal exposure | Fair-use review per clip; use reactions and edits, not raw audio/video segments |
| Black Boy Max subject unclear | Confirm subject before drafting; do not publish hooks until subject is locked |
| Volume burns out the team | Stagger cultural docs at 2-week intervals, not weekly — use the weeks between for catch-up |

## What This Calendar Does NOT Cover (and should)

- [ ] **Music licensing** for doc background music — separate workstream
- [ ] **Subtitles / translations** — high-leverage for global creator-economy audience
- [ ] **Podcast cut** of each long-form doc — second-life content
- [ ] **Paid amplification budget** for the series finale — this is the single asset worth promoting
- [ ] **Phase 3 pilot cohort** — sequencing into the D2C affiliate network is its own calendar

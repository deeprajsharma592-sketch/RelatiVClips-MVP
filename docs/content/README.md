## How to use this folder

1. **Adding a new doc?** Start from `00-template-cultural-doc.md`. Don't start from scratch.
2. **Drafting a hook angle?** Add it to `01-hook-angles.md` under the relevant doc section, not in the doc outline itself.
3. **Updating the calendar?** Edit `05-content-calendar.md`. Keep the week-by-week structure.

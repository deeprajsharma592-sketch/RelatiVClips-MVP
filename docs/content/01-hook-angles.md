# Short-Form Hook Angles — Tease Clips for the Cultural Documentary Series

> **Purpose:** Every long-form doc seeds 8–12 short-form clips for RelatiV's own channels. Each clip must be **self-contained** (the viewer doesn't need to have seen the doc) and **end with curiosity that the long-form resolves**, not with a hard sell.
>
> **Hook structure:** Pattern-interrupt visual (0–1s) → Tension line (1–4s) → Cliff / setup (4–15s) → Card (15–30s).
>
> **Per doc below: 3 hook angles**, scoped to different platform grammars (TikTok native, YouTube Shorts documentary-cut, Twitter/X quote-cut). More angles can be added per doc when drafting.

---

## PLAYBOI CARTI — Hook Angles

### Hook C1 — TikTok-native (15s)
- **Pattern-interrupt visual:** a slowed + reverb edit of a leaked Carti snippet, then SNAP-CUT to a SoundCloud comment from 2017 reading "this is the future of music"
- **Tension line:** `"Playboi Carti fans built a marketing department that never got paid."`
- **Cliff:** `"In 2017 they did something the labels didn't have a playbook for. It's a 12-minute doc — link in bio."`
- **Best platform:** TikTok (SoundCloud/Twitter-screenshot aesthetic performs well in music-TikTok)
- **CTA:** doc link in bio

### Hook C2 — YouTube Shorts documentary-cut (45s)
- **Pattern-interrupt visual:** a clean split: top half = a fan edit, bottom half = the original source clip. Sync them for 3 seconds.
- **Tension line:** `"Someone spent 6 hours cutting this. They didn't get paid. They made Carti a genre."`
- **Cliff:** `"We're going to map the entire network. The whole doc drops [DATE]."`
- **Best platform:** YouTube Shorts (longer-form docu-cut works better on Shorts than TikTok for this niche)
- **CTA:** subscribe + notify

### Hook C3 — Twitter/X quote-cut (text, 280c)
- **Format:** quote-tweet a 2018-era Carti snippet tweet with the overlay text `"This leak is from 2018. It got 40M streams because fans re-cut it 200 times. We made a doc about the network. [link]"`
- **Best platform:** X
- **CTA:** doc link

## Hook Production Checklist (per clip)
- [ ] Captions burned in (80%+ of short-form is watched on mute)
- [ ] First 1s has a movement / cut / number change (not a title card)
- [ ] Tension line is **1 sentence** that a stranger can repeat after one listen
- [ ] Cliff is **time-bound** ("drops Friday") not "soon"
- [ ] CTA matches platform grammar (link in bio / notify / subscribe)
- [ ] Vertical 9:16, no letterbox (Shorts/TikTok)
- [ ] Thumbnail A/B variant prepared if YouTube
- [ ] Legal review flag resolved (especially for Carti leaked audio, Adin Ross Twitch VODs)

# Documentary Outline Template — Cultural Breakdown

> Adapted from the GTM Case-Study Documentary structure (Bottleneck → Intervention → Payoff) for **cultural / creator-deep-dive** subjects. The "creator" in the original template becomes the **subject**. The "intervention" becomes the **fan-led / clipper-driven distribution mechanism**. The "payoff" becomes **cultural impact / market outcomes**.

Use this as a structural backbone. Each section lists **what to cover**, **how to source it**, and **what to leave blank for taste/voice decisions**.

---

## Section 1 — HOOK (0:00–0:30)
**Purpose:** Pattern-interrupt opener that earns the next 8–12 minutes.

- [ ] One cold-open visual that **embodies the cultural phenomenon** (not a title card)
- [ ] One spoken line that **states the thesis** as a tension, not a fact
  - Format: `"[Subject] [did counterintuitive thing]. The result was [unavoidable outcome]. Here's how."`
- [ ] One **hook retention target**: viewer must hit the 1:30 mark or the hook failed
- [ ] **Title card / thumbnail** — TBD (taste call)

## Section 2 — THE BOTTLENECK (0:30–3:00)
**Purpose:** Establish why the conventional path was supposed to fail for this subject.

- [ ] The **industry assumption** at the time (what "winning" was supposed to look like)
- [ ] The **specific gaps** in the subject's position vs. the assumption
  - Budget / team / mainstream access / release cadence / etc.
- [ ] **Why conventional marketing would have failed** them
- [ ] **Source assets:** archival interviews, era-specific media coverage, leaked DM screenshots (where ethically sourced)

## Section 3 — THE INTERVENTION (3:00–7:00)
**Purpose:** Document the *mechanism* — the fan-led / clipper-driven / decentralized distribution system that actually worked.

This is the **technical core** of the documentary and the part that doubles as RelatiV pipeline evidence. Be specific about **what fans/clippers actually did** mechanically:

- [ ] **Channel topology** — where the content lived (TikTok / Shorts / SoundCloud / Discord / Twitter / forums)
- [ ] **Asset manipulation** — re-edits, remixes, slowed+reverb, mashups, subtitles, side-by-sides
- [ ] **Distribution pattern** — multi-account posting, time-of-day tactics, cross-platform sequencing
- [ ] **Network effect** — how the asset looped back to the subject (e.g., artist reacting to clips, algorithm picking up the remix)
- [ ] **The RelatiV mirror** — explicit callout: *"This is exactly the pipeline RelatiV automates."* Optional but recommended for the alpha-→-Phase-2 bridge.

## Section 4 — THE PAYOFF (7:00–10:00)
**Purpose:** Quantify / dramatize the outcome.

- [ ] **Quantified impact** — chart-able metrics (views, chart position, cultural moments)
- [ ] **The "broken algorithm" moment** — specific clip / moment / date that crossed the threshold
- [ ] **Legacy** — what changed in the industry because of this playbook
- [ ] **Counterfactual** — "if the fan network hadn't done X, what would the timeline have looked like?" (optional, high-engagement segment)

## Section 5 — THE TAKEAWAY (10:00–11:30)
**Purpose:** Make it useful to the viewer (creators / brands watching).

- [ ] **3–5 transferable principles** the viewer can apply to their own distribution
- [ ] **The RelatiV hook** — tie back to the product without being salesy
- [ ] **CTA** — depends on funnel stage (subscribe / book demo / watch next doc)

## Section 6 — POST-PUBLISH SHORT-FORM FANOUT
Each doc seeds **5–10 short-form clips** for RelatiV's own channels. Use the `01-hook-angles.md` workbook for angle templates.

---

## Sourcing Checklist (per documentary)
- [ ] Primary sources: subject's own posts, interviews, leaked-but-publicly-circulated audio/video
- [ ] Secondary: cultural commentary, journalist longform, academic analyses (where available)
- [ ] Tertiary: fan edit archives (Twitch VODs of fan reactions, archived subreddits, Discord screenshots — verify timestamps)
- [ ] **Legal review flag:** any clip >5s from copyrighted source needs fair-use check before publishing
- [ ] **Cultural-sensitivity review flag:** when covering Black creators / subcultures, get a community-adjacent reviewer (don't ship with just the founder's read)

---

## Length Targets
- **Long-form:** 10–14 minutes (YouTube sweet spot for case-study retention)
- **Mid-form cut:** 4–6 minute "Twitter/X cut" for cross-posting
- **Short-form seeds:** 8–12 clips of 15–60s (RelatiV's core product proves this is the right format)

---

## Anti-Patterns (don't do this)
- ❌ Treating the subject as a "hack" to be reverse-engineered for empty growth-hacking content
- ❌ Speculating about motives without sources
- ❌ Calling it a "strategy" in a way that disrespects the artistry / cultural labor
- ❌ Burning the subject's name in a clickbait thumbnail the subject would find disrespectful
- ❌ Missing the cultural moment: if the doc is timely (release anniversary, comeback, scandal), the publish date is the thesis
